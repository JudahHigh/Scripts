Title: put_files_in_dirs.py
Author: Judah High

Description: First looks for a directory the user specifies in the current
working directory. It will copy any files, along with their respective
executable permissions if possible to each sub-directory within the current
working directory.

Use: Within the working directory of choice, pass the script the name
of the directory containing files to copy.

>> python put_files_in_dirs.py src_dir
