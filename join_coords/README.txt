Title: join_coords.py
Author Judah High

Description: this script simply joins two Gaussian molecular input files
(.com extensions) into one. It does not check for overlapping atoms, the
user must do this.

Use: Pass two .com files to script as arguments,

>> python join_coords.py a.com b.com

The above command results in a new input file called joined_a_b.com
being creating.
