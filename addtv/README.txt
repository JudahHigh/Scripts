Title: addtv.py
Author: Judah High

Description: This is a simple script that adds user defined translation vectors to a pre-made
.com input file for Gaussian. The user must hardcode the translation vectors Vx, Vy,
and Vz into the script source code where defined.

If translation vectors are already defined the script will not replace them and
will exit.

Use: to use call script and pass the input .com as an argument.

>> python addtv.py some_molecule.com
